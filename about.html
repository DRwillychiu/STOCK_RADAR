About Stock Radar

個人化台股研究儀表板。設計目的：當老闆隨口問起一檔股票時，能在 5 分鐘內掌握公司產業定位、營收結構、轉型動向、近期動態，並提出有見地的觀察。

01 三層架構
研究層 · Claude SKILL stock-research
資料層 · GitHub Actions + JSON 每日 07:00 自動更新
展示層 · GitHub Pages 純靜態 HTML / CSS / JS
02 資料來源
FinMind 主源 · 財報、月營收、股價
Yahoo Finance 副源 · 即時報價、市值
公開資訊觀測站 MOPS 重訊、法說會
Goodinfo 同業比較、產業歸類
03 使用流程
  1. 老闆問股票 → 在搜尋框輸入股號或公司名
  2. Watchlist 內的股票 → 立即顯示完整詳情頁
  3. Watchlist 外的股票 → 顯示提示，請 Claude 跑 SKILL 深入研究
  4. 研究結果 commit 到 repo → 隔天自動納入排程
04 資料一致性燈號
綠燈 兩個以上資料源完全一致
黃燈 資料源有小差異或單源資料
紅燈 資料源有明顯差異，需人工確認
DISCLAIMER · 免責聲明
  • 本系統為個人研究工具，所有資料僅供參考
  • 不構成任何投資建議、買賣訊號或財務顧問意見
  • 資料準確性以官方公告（TWSE、TPEX、MOPS）為準
  • 使用者應自行判斷投資價值與風險，獨立做出投資決策
